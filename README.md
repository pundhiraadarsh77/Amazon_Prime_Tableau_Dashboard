# Amazon Prime Analytics Dashboard | Tableau

An interactive Tableau dashboard built to analyze Amazon Prime's content library and uncover meaningful business insights. This project explores content distribution, genre popularity, production trends, IMDb ratings, and cast information through interactive visualizations and dashboards.

---

## Project Overview

The objective of this project is to analyze Amazon Prime's content catalog and present key insights using Tableau. The dashboard enables users to explore the platform's library through interactive filters and visualizations, helping identify trends in content production, genres, countries, ratings, and cast appearances.

---

## Business Objective

The goal of this dashboard is to transform raw Amazon Prime data into actionable business insights that can support content strategy and decision-making by answering questions such as:

- What is the distribution of Movies vs TV Shows?
- Which genres dominate the platform?
- Which countries contribute the most content?
- How has content production evolved over time?
- Who are the most frequently appearing actors and directors?
- What are the overall IMDb rating trends?

---

## Dataset

The analysis is based on Amazon Prime content and cast datasets.

### Data Preparation
- Merged title and cast datasets using Python (Pandas)
- Cleaned missing and inconsistent values
- Removed duplicates
- Created derived fields for analysis
- Prepared a dashboard-ready dataset for Tableau

# Dashboard Overview

The Tableau Story consists of two interactive dashboards.

## Dashboard 1 — Title Overview

Features:

- KPI Cards
  - Total Titles
  - Total Movies
  - Total TV Shows
  - Average IMDb Rating
  - Average Runtime

- Movies vs TV Shows Distribution
- Titles Released Over Time
- Top 10 Genres
- Top 10 Production Countries
- Interactive Filters

---

## Dashboard 2 — Cast Analysis

Features:

- KPI Cards
  - Total Actors
  - Total Directors
  - Total Cast Appearances
  - Average IMDb Rating

- Top 10 Actors
- Top 10 Directors
- Role Distribution
- Genre Distribution by Cast
- Country Distribution
- Interactive Dashboard Actions

---

# Key Insights

- Movies significantly outnumber TV Shows on Amazon Prime.
- Drama and Comedy are among the most common genres.
- The United States contributes the highest number of titles.
- Content production has grown rapidly in recent years.
- A small group of actors appears across a large number of titles.
- IMDb ratings are concentrated around the mid-to-high rating range.
